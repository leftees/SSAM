Simple Site Audit Multisite - VirusTotal Scanner


Instructions:

In order to use this utility, you will need to register at VirusTotal.com This is simply to obtain a VirusTotal API key of your own. Scans can also be run from their site.

Registration and usage is free of charge. Once you have registered, you will find your API key in your profile on the 'API' tab.

Please be aware that the public API can only be used for non-commercial purposes. Remember also that it is restricted to 4 requests of any nature per minute. Stick to the rules and you should not experience any problems.
